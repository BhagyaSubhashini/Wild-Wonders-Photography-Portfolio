# Wild Wonders Photography Portfolio

Welcome to the **Wild Wonders Photography** portfolio website! This project showcases the captivating work of a passionate wildlife and nature photographer, dedicated to capturing the soul of the wilderness‚Äîone breathtaking frame at a time. Built entirely with HTML and CSS, it's a clean, responsive single-page application (SPA-like navigation) that highlights services, stunning portfolios, gear, media embeds, testimonials, and easy contact options.

## üìã Features
- **Responsive Design**: Adapts seamlessly to desktop, tablet, and mobile using CSS media queries.
- **Navigation**: Smooth single-page scrolling with anchor links for Home, About, Services, Portfolio, Gear, Media, Testimonials, and Contact.
- **Sections Breakdown**:
  - **Home**: Hero section with logo, title "Capturing the Essence of Untamed Nature", and immersive nature background.
  - **About**: Personal bio with a profile photo and quote about passion for wildlife photography, silent jungles, vibrant ecosystems, and conservation.
  - **Services**: Three cards for "Wildlife Photo Shoots" (custom sessions in forests/jungles/parks), "Nature Documentary Creation" (short docs on nature/conservation), and "Workshops & Mentorship" (hands-on for aspiring wildlife photographers).
  - **Portfolio**: Grid gallery titled "Photography Journey" with sample images (e.g., landscapes, flowers, sunsets) and photographer credits.
  - **Gear**: Table listing equipment like Canon EOS R5 camera (high-res wildlife), Canon RF 100-500mm lens (long-range shots), DJI Mavic 3 drone (aerial captures), and Manfrotto Befree tripod (stable long exposures).
  - **Media**: Embedded YouTube/BBC-style videos (e.g., "A Day In The Life", "Best Cameras for Wildlife", "Costa Rica Wildlife") in a card grid.
  - **Testimonials**: Four quote cards from clients like Amelia Rivers ("Absolutely stunning wildlife photography..."), Daniel Moss, Sophia Lee, and Marcus Bennett.
  - **Contact**: Simple form with name, email, message fields, submit button, and social icons (Facebook, Instagram, Twitter, YouTube).
- **Styling**: Dark/green/teal color scheme inspired by nature, with fern/leaf motifs, glowing effects, and high-contrast text for readability.
- **No JS**: Pure HTML/CSS for lightweight performance‚Äîideal for static hosting.
